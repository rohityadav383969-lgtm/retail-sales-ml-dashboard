# 📌 Project Overview

The **Retail Sales Machine Learning Dashboard** is an interactive data science project built using **Python, Machine Learning, and Streamlit**.

This dashboard analyzes retail sales data, performs exploratory data analysis (EDA), compares machine learning models, and predicts future sales using trained regression models.

The goal of this project is to demonstrate practical knowledge in:

* Data Cleaning
* Data Analysis
* Machine Learning
* Data Visualization
* Dashboard Development

---

# 🎯 Project Objectives

The objectives of this project are:

* Analyze retail sales dataset
* Perform exploratory data analysis (EDA)
* Train multiple machine learning models
* Compare ML model performance
* Build an interactive dashboard for predictions

---

# 🤖 Machine Learning Models Used

This project compares the performance of multiple machine learning algorithms:

* Linear Regression
* Random Forest Regressor
* XGBoost Regressor

The dashboard evaluates the models using **R² Score** and identifies the **best performing model**.

---

# 📊 Dashboard Features

## 🏠 Home Page

Displays key statistics of the dataset including:

* Total Records
* Unique Product Categories
* Average Customer Age

---

## 📂 Dataset Page

Displays the complete dataset in tabular format.

This allows users to explore the raw data used for analysis.

---

## 📈 Exploratory Data Analysis (EDA)

The EDA section visualizes the dataset using charts such as:

* Sales by Product Category
* Gender Distribution
* Age Distribution

Charts are built using **Matplotlib, Seaborn, and Plotly**.

---

## 🤖 Model Comparison

This section compares machine learning models including:

* Linear Regression
* Random Forest
* XGBoost

The system automatically identifies the **best performing model**.

---

## 🔮 Sales Prediction

Users can input values such as:

* Age
* Quantity
* Price per Unit

The trained machine learning model predicts the **expected sales amount**.
